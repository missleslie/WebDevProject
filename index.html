Week 10

Published on December 16th 2014

For my second style sheet I wanted the webpage to look more modern and have a fade in effect when the page is first opened. I struggled with this for quite a while as quite a few website tutorials recommended that I use JavaScript.

I used the [3] Stack Overflow Questions and Answers website to help me with this. While experimeenting with thius teechnique I learnt that different browsers require their own different type of animation CSS property. For example, the browsers Google Chrome and Safari use '-webkit', Firefox uses '-moz' and Internet Explorer uses '-ms'. I learnt more about browser compatability types from the [4] Mozilla Developer Network. So after discovering this I ensured that if I was to have any animation in my webpage, that I should write code that would ensure that it would work on all type of browsers.
